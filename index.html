Download extension (.zip) or view the source on GitHub
How to install (2 minutes)
1
Download the .zip above and unzip it. The extension lives in the clipper folder inside.
2
Open a new tab and go to chrome://extensions
3
Turn on Developer mode (toggle, top-right)
4
Click Load unpacked → select the clipper folder
5
Pin it, then in the extension's Settings paste this app's URL so clips sync back here
Then just clip. On any job posting, hit the floating Clip button (or the toolbar icon) and it lands in your Wishlist automatically.
